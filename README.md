
# Analytics Portfolio

Data Analytics Portfolio – FMCG Sales Analytics and Fintech Remittance Analytics

This repository contains analytics projects focused on:

1. FMCG Sales Analytics (inspired by Unilever Secondary Sales Analytics)
2. Fintech Remittance Analytics (Cross-border Transaction Insights)

The goal of this portfolio is to demonstrate data-driven decision making using Python and Business Analytics techniques.

Projects will include:
- KPI analysis
- Customer behavior insights
- Sales and transaction trends
- Growth opportunity identification
