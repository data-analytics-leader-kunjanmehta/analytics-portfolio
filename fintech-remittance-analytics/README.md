# Fintech Remittance Analytics Project

This project analyzes cross-border transaction data inspired by fintech remittance platforms.

Objectives:
- Understand transaction patterns
- Analyze customer transaction behavior
- Identify high-performing transfer corridors
- Measure customer retention

Key KPIs:
- Transaction Volume
- Active Customers
- Average Transaction Value
- Customer Retention Rate
- Corridor Performance

Tools to be used:
- Python
- Pandas
- Data Visualization
