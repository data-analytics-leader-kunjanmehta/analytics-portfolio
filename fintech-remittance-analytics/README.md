# Fintech Remittance Analytics Project

This project analyzes cross-border transaction data inspired by fintech remittance platforms.

**Objectives:**
- Understand transaction patterns
- Analyze customer transaction behavior
- Identify high-performing transfer corridors
- Measure customer retention

**Key KPIs:**
- Transaction Volume
- Active Customers
- Average Transaction Value
- Customer Retention Rate
- Corridor Performance

**Tools to be used:**
- Python
- Pandas
- Data Visualization

**Funnel Conversion Analysis**

The customer journey was analyzed using a three-stage funnel:

1. User signup

2. KYC verification

3. Transaction completion

Conversion rates between stages were calculated to identify customer drop-off points and potential growth opportunities.
