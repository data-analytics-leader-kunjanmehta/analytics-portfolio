# FMCG Sales Analytics Project

This project analyzes FMCG sales data inspired by secondary sales analytics in the FMCG industry.

**Objectives:**
- Analyze product performance
- Evaluate distributor sales performance
- Identify regional sales trends
- Detect potential stock-out risks

**Key KPIs:**
- Total Sales Revenue
- Sales Growth
- Product Category Performance
- Distributor Performance
- Inventory vs Demand

**Tools to be used:**
- Python
- Pandas
- Data Visualization


**Business Context**

This project simulates a sales analytics use case in the FMCG industry similar to analytics work performed for large consumer goods companies such as Unilever.

The objective is to analyze sales performance across regions, products, and distributors and generate insights that support business decision-making.

**Business Problem**

Sales organizations often face challenges such as:

Identifying high-performing regions

Monitoring product performance

Managing distributor channel performance

Forecasting future demand

This project demonstrates how analytics can be used to solve these challenges.

**Dataset**

The dataset includes:

Sales date

Region

Distributor

Product category

Sales revenue

Inventory levels

**Analytics Performed**

The following analyses were conducted using Python in Google Colab:

**KPI Analysis :**

Total sales

Sales by region

Sales by product

**Visualization**

Regional sales performance

Product performance charts

**Trend Analysis**

Sales trends over time

**Segmentation Analysis**

Distributor performance comparison

**Predictive Analytics**

Sales forecasting using Linear Regression

**Business Insights**

Key insights from the analysis include:

Certain regions contribute higher sales revenue.

Specific product categories drive stronger performance.

Distributor segmentation highlights top-performing sales channels.

Trend analysis helps detect sales fluctuations.

Predictive modeling provides early signals for demand planning.
